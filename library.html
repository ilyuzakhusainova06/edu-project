Библиотека инструментов

Каталог цифровых инструментов для педагога начальной школы

Презентации

Microsoft PowerPoint
Классический инструмент для создания презентаций с анимациями, триггерами и интерактивными элементами
Google Slides
Облачные презентации с возможностью совместной работы в реальном времени
Prezi
Нелинейные презентации с эффектом масштабирования и динамичной навигацией

Графика

Canva
Онлайн-редактор с шаблонами для создания инфографики, плакатов, карточек и социальных медиа
GIMP
Бесплатный растровый графический редактор с профессиональными возможностями
Paint.NET
Простой и интуитивный графический редактор для базовой обработки изображений

Онлайн-платформы

LearningApps
Платформа для создания интерактивных упражнений: викторины, кроссворды, пазлы
Wordwall
Создание игровых заданий с множеством шаблонов: колесо фортуны, лабиринт, гонка
Kahoot!
Платформа для проведения интерактивных викторин и опросов в реальном времени
Padlet
Виртуальная доска для совместной работы, сбора идей и организации материалов

Google Workspace
Набор облачных инструментов: Документы, Таблицы, Формы, Класс — для организации учебного процесса
Яндекс 360
Российские облачные сервисы: Диск, Документы, Формы — для хранения и совместной работы
Облако Mail.ru
Облачное хранилище для размещения и обмена учебными материалами
