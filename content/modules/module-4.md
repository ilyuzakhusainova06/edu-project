---
title: Интерактивные платформы
number: 4
description: LearningApps, Wordwall и аналоги — создание интерактивных заданий и их интеграция в урок.
goals:
  - Познакомиться с платформами для создания интерактивных заданий
  - Научиться создавать ЦОР на базе LearningApps и аналогов
  - Освоить интеграцию интерактивных заданий в структуру урока
theory: >-
  LearningApps.org — бесплатная платформа для создания интерактивных упражнений:
  викторины, кроссворды, пазлы, задания на сопоставление и классификацию.

  Альтернативы: Wordwall, Quizlet, Kahoot!, Padlet. Каждая платформа имеет свои
  преимущества в зависимости от типа задания и возраста учеников.
pedagogy: >-
  Интерактивные задания можно использовать на разных этапах урока: мотивация
  (вводная викторина), закрепление (кроссворд по теме), контроль (тест), рефлексия
  (интерактивный опрос).

  Важно не перегружать урок цифровыми инструментами — они должны дополнять, а не
  заменять живое общение с учителем.
practice: >-
  Разработка ЦОР для урока в начальной школе:

  - Создать 3 интерактивных упражнения на LearningApps

  - Указать этап урока для каждого упражнения

  - Подготовить краткую инструкцию для ученика
reflection:
  - Какие типы интерактивных заданий наиболее эффективны для младших школьников?
  - Как найти баланс между цифровыми и традиционными методами обучения?
